# 将棋RPG (shogi_rpg)

将棋の対局そのものをコアバトルシステムとして扱うブラウザ向けRPG。
物語の進行に応じて、使用可能な戦形・アイテム・スキルが増えていき、
敵（将棋AIエンジン）も段階的に強くなっていく。

## コンセプト

- プレイヤーはRPGを進めながら将棋の実力とゲーム内リソース（戦形・アイテム・スキル）の両方で強くなっていく
- 「装備」や「レベルアップ」に相当する要素として、将棋の戦形・駒落ち・ヒント・待った・敵デバフなどをアイテム/スキル化する
- 全て無料の静的ホスティングで配信するため、**将棋AIの思考はすべてクライアント（ブラウザ）側で実行**する

## 技術スタック

| レイヤー | 技術 | 役割 |
|---|---|---|
| 演出・進行管理 | ティラノスクリプト | ノベルパート、章の進行、フラグ管理、UI遷移 |
| 対局UI | JavaScript（既存ライブラリ＋自作） | 盤面表示、指し手入力、成り選択、持ち駒表示 |
| 将棋AIエンジン | YaneuraOu.wasm（USIプロトコル） | ブラウザのメインスレッドから直接実行（詳細は `docs/CLAUDE.md`）、章ごとに強さ・評価関数を変更 |
| 開発・検証用ツール | cshogi / python-shogi | 定跡データ作成、AI強さのチューニング検証、棋譜処理（本番では非使用） |
| セーブ | localStorage + 復活の呪文（文字列） | 端末間引き継ぎ、データ消失時のバックアップ |

## ホスティング方針

無料の静的ホスティング（GitHub Pages等）を利用する前提のため、サーバーサイドの処理は極力持たない。
将棋AIの推論・ルール処理・セーブデータ管理はすべてクライアントサイドで完結させる。

## 対応環境

- **必須**: PC・Google Chrome
- **努力目標**: スマートフォン（Chrome等）

## ライセエンス・素材に関する留意事項

- 将棋AIエンジン（YaneuraOu）本体は **GPLv3**。自作コードと結合して配布する場合、エンジン部分のソース開示義務が発生する点に留意
- 評価関数（NNUE）ファイルは自由配布のもの（リゼロ評価関数など）のみ使用可。水匠など支援者限定配布のファイルは同梱不可
- 駒・盤画像はフリー素材（CC BY-SA等、クレジット表記必須なものを含む）を優先し、不足分は自作する
