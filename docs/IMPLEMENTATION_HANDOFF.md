- 7つの村、夜古火姫、名前入力、3段階難易度、地点選択式探索、店、棋貨、宝箱、
  サブクエスト、ボスへの直接挑戦、初回勝利経験値を整理した本編ゲーム設計

  叩き台である`data/world.json`

- 13冊の定跡書と矢印ガイド手順の叩き台である`data/books.json`
- 名前、呼称、経験値、棋貨、敵撃破、定跡書、宝箱、講座、クエストを保存する
  version 2セーブ状態と、`SR2-`形式の復活の呪文の途中実装

## 4. 次に行う作業

### P0: 既存機能を壊さない状態へ戻す

1. 現行M4シナリオの敵IDを新マスタへ合わせるか、移行期間用の稽古相手を残し、対局開始を復旧する
2. version 2セーブを完成させ、version 0・1からの移行、破損復旧、localStorage、
   version 1復活の呪文の読込互換をテストする
3. version 2の復活の呪文について、文字数上限、UTF-8名、チェックサム、カタログ更新時の扱いを確定する
4. 既存105テストをすべて成功させてから、新しい進行テストを追加する
5. `world.json`と`books.json`へ重複ID、参照先ID、範囲、必須項目、USI合法手の検証を追加する

### P1: 第1章を縦に完成させる

1. `world.json`を読む進行モジュールを作り、初回勝利だけの経験値・棋貨報酬、レベルアップ、
   宝箱、店、サブクエスト、ボス撃破による次章解禁を実装する
2. 名前・呼称・難易度入力から、始まりの町の地点選択、村人戦、村長戦、勝利後会話まで接続する
3. ボスへ到着直後から挑戦でき、通常対局を後回しにでき、敗北後も同じ村で育成できることをテストする
4. 主人公は立ち絵と発話を持たず、選択肢と1〜2文の内心だけで表現する
5. ここで本編外枠を、ティラノ拡張で実装するか、対局iframeを維持したデータ駆動SPAへ移すか確定する。
   推奨は、既存対局iframeとメッセージ契約を再利用し、村・会話・報酬画面をJSON駆動にする構成

P1の完了条件は、仮素材のまま第1章を開始から村長撃破まで通して遊べ、再読込後も進行が維持され、
自動テストとHostingビルドが成功することとする。

### P2: 学習機能を完成させる

1. 盤上の矢印、強調マス、解説吹き出しを共通ガイド表示として実装する
2. 将棋の基本、頭金、棒銀、四間飛車、美濃囲い、右四間飛車、エルモ囲いを章講座へ接続する
3. 誤手を説明して自動で戻すチュートリアルと、8〜16手・2〜3分岐の定跡書再生を実装する
4. 入手済み定跡書を定跡集で再閲覧し、許可された本だけ対局中ガイドへ出せるようにする
5. 全定跡手順を合法手検証し、初心者向け解説を人手でレビューする

### P3: 第2〜7章とバランスを完成させる

1. 第1章の完成パターンを残り6章へ展開し、25対局をすべて接続する
2. 右四間＋エルモ囲い村は公開可能な内蔵評価で完成させ、振電3を必須にしない
3. 初心者の通常攻略8〜12時間と熟練者のボス直行約30分を、計測ログに基づいて調整する
4. 3段階難易度で進行報酬を変えず、対局の思考量と候補手品質だけが変わることを確認する

### P4: 素材差し替え可能な完成形にする

1. キャラクター、表情、背景、イベント絵、BGM、効果音、任意ボイスを素材IDから参照する
2. 必要サイズ、形式、ループ点、ファイル名、使用場面を素材台帳へ列挙する
3. すべての素材枠へ再配布可能なプレースホルダーを配置し、ファイル交換だけで更新できることを確認する
4. 未使用のティラノサンプル素材を配布物から除外し、GPL対応ソースと第三者通知を完成させる

### P5: リリース検証

1. PC Chromeで新規開始、通常攻略、ボス直行、敗北・再挑戦、セーブ移行、全章完走を確認する
2. スマートフォン実機で主要画面、WASM速度、メモリ、タップ操作を確認する
3. 自動テスト、Hostingビルド、差分検査、ライセンス監査をすべて成功させる
4. 明示的な依頼を受けた場合だけFirebase本番へデプロイし、公開URLをスモークテストする

## 5. 変更時に守ること

- 将棋AI、ルール判定、進行、セーブはすべてブラウザ内で完結させる
- `data/*.json`を本編コンテンツの正とし、画面やシナリオへ同じ値を重複記述しない
- 通常の村人対局をボス挑戦の必須条件にしない
- 経験値は各相手への初回勝利時だけ付与する
- 振電3は作者から明示的な再配布許可を得るまで追加しない
- `dist/`を直接編集しない
- 本番デプロイはユーザーから明示的に依頼された場合だけ行う
