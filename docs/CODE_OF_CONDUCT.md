# 行動規範 (CODE OF CONDUCT)

本プロジェクトは、将来的に外部からのコントリビューションを受け入れる可能性があるOSSプロジェクトです。
参加者全員が快適に開発・議論できるよう、以下の規範を定めます。

## 基本方針

- 互いを尊重し、建設的なコミュニケーションを心がける
- 技術的な議論は歓迎するが、人格攻撃・侮辱的な言動は禁止する
- 将棋の実力・プログラミングの熟練度に関わらず、初心者の質問や提案を歓迎する

## 禁止事項

- 差別的な言動（性別・国籍・年齢・障がい・信仰等に基づくもの）
- 誹謗中傷、嫌がらせ、ストーキング行為
- 他者のプライバシーを侵害する行為（本人の同意のない個人情報の公開等）
- スパム行為、無関係な宣伝

## 違反時の対応

- Issue・PR・コメント等で違反が確認された場合、注意・編集・削除等の対応を行うことがあります
- 悪質な場合、プロジェクトへの参加をお断りすることがあります

## 適用範囲

本規範は、GitHubリポジトリ上のIssue・Pull Request・Discussion、および本プロジェクトに関連する
その他のコミュニケーションチャネル全般に適用されます。

## 連絡先

問題を発見した場合、リポジトリのIssueまたは管理者へ直接連絡してください。
