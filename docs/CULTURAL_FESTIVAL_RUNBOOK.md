# 文化祭展示 運用メモ

## 開始前

1. 各端末を電源へ接続し、Chromeで公開URLを開く。
2. 盤面と「あなたの番です」が表示されることを確認する。
3. 1手指して、AIが応手することを確認する。

## 通常の交代

- 対局終了後は「同じ条件でもう一度」を押す。
- 対局途中で交代するときは「最初から」を押し、確認画面で続行する。

## 問題が起きたとき

1. 「再試行」が表示された場合は、Wi-Fi接続を確認してから押す。
2. 反応しない場合は `Ctrl+F5` で再読み込みする。スマートフォンではタブを閉じて開き直す。
3. 直らない場合はChromeを終了して起動し直す。
4. それでも直らない端末は展示から外し、別端末を使用する。

「オフラインです」と表示されても、すでに開始済みの対局は続行できる。通信が戻るまで
再読み込みは行わない。

## 終了時

- 展示用タブを閉じる。
- 端末の通常のスリープ設定が戻っていることを確認する。
